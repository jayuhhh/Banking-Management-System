# Banking-Management-System
A robust, console-based retail banking application built in Python using Object-Oriented Programming (OOP) principles. The system simulates a secure, real-time banking terminal by automating fundamental financial workflows, validating transactions, and maintaining a rolling ledger.

Key Features:
Account Lifecycle Management: Securely creates user accounts with dynamic 6-digit identification numbers and sets mandatory baseline deposit thresholds.

Automated Financial Operations: Processes real-time asset injections (deposits) and fund liquidations (withdrawals) with built-in input validation and overdraft protection.

Dynamic Statement Ledger: Implements a comprehensive transactional history tracker that auto-generates structured statements detailing timestamps, transaction types (Debit/Credit), transaction values, and rolling balances.

Encapsulation & Validation: Engineered with robust input handling to filter structural runtime anomalies, non-positive financial values, or unauthorized database access.
